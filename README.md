# 📚 Bookstore Management System

## 🔹 Overview
The **Bookstore Management System** is a dynamic web-based application built using HTML, CSS, and JavaScript. It allows users to manage books efficiently by adding, searching, and deleting entries in real time.

The project demonstrates core front-end development concepts such as DOM manipulation, event handling, and client-side data storage.

---

## 🔹 Features
* ➕ Add books with title and author  
* ❌ Delete books instantly  
* 🔍 Search books in real-time  
* 💾 Persistent storage using **localStorage**  
* ⚡ Dynamic UI updates without page reload  
* 🎨 Clean and user-friendly interface  

---

## 🔹 Technologies Used
* HTML  
* CSS  
* JavaScript  

---

## 🔹 How to Run
1. Clone or download this repository  
2. Open the project folder  
3. Run the `index.html` file in your browser  

---

## 🔹 How It Works
1. User enters book details (title and author)  
2. Clicks on **Add Book**  
3. Book is displayed instantly on the screen  
4. Data is stored in **localStorage**  
5. Users can search or delete books dynamically
